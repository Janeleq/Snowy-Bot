# Snowy-Bot
A discord pet bot made for friends<br/>
Made with Discord.PY

Does simple tasks like 
<ul>
<li>About me</li>
<li>Greeting</li>
<li>Attacking</li>
<li>Treats</li>
<li>Displaying pictures</li>
<li>Checking of pet closeness</li>
</ul>
